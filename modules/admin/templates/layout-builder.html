CLI Commands

You can also manage layouts via CLI:

# List all layouts
node index.js layout:list

# View sections for a content type
node index.js layout:sections page

# Add a section
node index.js layout:add-section page two_column

# Add a block to a section
node index.js layout:add-block page <sectionUuid> content <blockId>

# Remove a section
node index.js layout:remove-section page <sectionUuid>

# View statistics
node index.js layout:stats
