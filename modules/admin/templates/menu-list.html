About Menus

What are Menus?

Menus are collections of navigation links displayed on your site. Common menus include Main Navigation, Footer Links, and User Account menus.

Menu Items

Each menu contains items (links) that can be organized hierarchically with parent/child relationships for dropdown navigation.

Theme Integration

Reference menus in your theme templates using their machine name to display navigation anywhere on your site.
