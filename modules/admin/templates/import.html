CLI Import

Import can also be done via command line:

# Preview what would be imported (dry run)
node index.js import:content backup.json --dry-run

# Import content (skip existing)
node index.js import:content backup.json

# Import content (overwrite existing)
node index.js import:content backup.json --overwrite

# Full site import (content only by default)
node index.js import:site site-backup.json --dry-run

# Full site import with config
node index.js import:site site-backup.json --import-config

Export Format

The import expects JSON in this format:

{
  "version": "0.0.20",
  "exported": "2024-01-15T12:00:00Z",
  "content": {
    "greeting": [
      { "id": "123-abc", "type": "greeting", "name": "...", ... },
      { "id": "456-def", "type": "greeting", "name": "...", ... }
    ],
    "user": [ ... ]
  },
  "config": {
    "site": { ... },
    "modules": { ... }
  },
  "media": [ ... ]
}
