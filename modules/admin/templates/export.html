CLI Export

Export can also be done via command line:

# Export all content
node index.js export:content > backup.json

# Export specific types
node index.js export:content greeting user > users-and-greetings.json

# Export to file directly
node index.js export:content --output=backup.json

# Full site export
node index.js export:site --output=site-backup.json
