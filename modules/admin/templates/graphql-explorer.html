GraphQL Endpoint

Send GraphQL queries to the following endpoint:

POST /graphql

Example Request

curl -X POST "http://localhost:3000/graphql" \
  -H "Content-Type: application/json" \
  -d '{"query": "{ greetings(limit: 5) { id name message } }"}'

Example Response

{
  "data": {
    "greetings": [
      { "id": "abc123", "name": "Hello", "message": "Welcome!" }
    ]
  }
}
