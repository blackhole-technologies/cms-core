About Content Locking

How it works:

  • When a user opens content for editing, a lock is acquired
  • Other users cannot edit the same content while it's locked
  • Locks automatically expire after {{timeout}} seconds of inactivity
  • A {{gracePeriod}}-second grace period allows the original user to reclaim an expired lock
  • Admins can force-release any lock if needed
