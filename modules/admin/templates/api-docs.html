Getting Started

The CMS API supports multiple versions for backwards compatibility. You can specify the API version in several ways:

URL Prefix (Recommended)

GET /api/v2/content/article

Request Header

GET /api/content/article
X-API-Version: 2

Accept Header

GET /api/content/article
Accept: application/vnd.cms.v2+json

Query Parameter

GET /api/content/article?api_version=2

Response Formats

v1 Response (Legacy)

  "items": [...],
  "total": 25
}

v2 Response (Current)

{
  "data": [...],
  "meta": {
    "total": 25,
    "page": 1,
    "limit": 20,
    "pages": 2
  }
