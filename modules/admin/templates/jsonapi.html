API Reference

List Resources

GET {{basePath}}/{type}

Returns a paginated list of resources.

Query Parameters

page[offset] Pagination offset (default: 0)
page[limit] Items per page (default: 20, max: 100)
sort Sort field, prefix with - for descending
filter[field] Filter by field value
filter[field][op] Filter with operator (eq, ne, gt, gte, lt, lte, contains)
include Comma-separated relationships to include
fields[type] Sparse fieldset - only return specified fields

Get Single Resource

GET {{basePath}}/{type}/{id}

Returns a single resource by ID.

Create Resource

POST {{basePath}}/{type}

Creates a new resource. Request body must be a JSON:API document.

{
  "data": {
    "type": "article",
    "attributes": {
      "title": "My Article",
      "body": "Content here..."
    }
  }
}

Update Resource

PATCH {{basePath}}/{type}/{id}

Updates an existing resource.

Delete Resource

DELETE {{basePath}}/{type}/{id}

Deletes a resource. Returns 204 No Content on success.

Example Requests

List with pagination and sorting

GET {{basePath}}/article?page[limit]=10&sort=-created

Filter by field

GET {{basePath}}/article?filter[status]=published

Include relationships

GET {{basePath}}/article/123?include=author,tags

Sparse fieldsets

GET {{basePath}}/article?fields[article]=title,created
