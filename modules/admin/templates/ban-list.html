About IP Banning

IP bans prevent access from specific IP addresses or CIDR ranges. Banned IPs receive a 403 Forbidden response.

  • Single IP: 192.168.1.100
  • CIDR Range: 10.0.0.0/8 (entire subnet)
  • Permanent Bans: Never expire automatically
  • Temporary Bans: Automatically expire after the specified duration

Bans are checked before all other request processing and apply to all endpoints.
