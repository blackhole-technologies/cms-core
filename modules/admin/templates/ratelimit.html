Rate Limit Configuration

Configure rate limiting in config/site.json:

{
  "rateLimit": {
    "enabled": true,
    "login": { "points": 5, "duration": 60, "blockDuration": 300 },
    "api": { "points": 100, "duration": 60 },
    "admin": { "points": 60, "duration": 60 }
  }
}

points: Max requests allowed in window
duration: Time window in seconds
blockDuration: How long to block after exceeding limit
