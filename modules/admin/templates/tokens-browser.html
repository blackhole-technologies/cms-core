How to Use Tokens

Tokens are placeholders that get replaced with dynamic values. Click any token below to copy it to your clipboard.

Examples:

  • [site:name] - Replaced with your site name
  • [current-user:name] - Replaced with the logged-in user's name
  • [content:title] - Replaced with the content title
  • [date:medium] - Replaced with the current date

Chained Tokens:

You can chain tokens to access nested properties:

  • [content:author:name] - Author's name from content
  • [content:author:email] - Author's email from content

Raw Modifier:

By default, tokens are HTML-escaped for security. Use :raw to disable escaping:

  • [content:body:raw] - Unescaped HTML content
