About Blocks

What are Blocks?

Blocks are reusable content chunks that can be placed in regions throughout your site. Examples: sidebars, footers, headers, widgets.

Block Types

Content blocks contain custom content. System blocks are generated by modules (like search, navigation, user menus).

Regions

Regions are areas in your theme where blocks can be placed. Common regions: header, sidebar, footer, content.

Visibility

Control where blocks appear using visibility conditions based on URL paths, content types, or user roles.
