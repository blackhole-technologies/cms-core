Plugin Auto-Reload

Auto-reload mode: {{autoReloadMode}}

  • node index.js plugins:autoload on - Automatically reload on file change
  • node index.js plugins:autoload off - Ignore file changes
  • node index.js plugins:autoload prompt - Log changes, require manual reload
  • node index.js plugins:watch - Watch for changes in real-time
  • node index.js plugins:reload-changed - Reload all changed plugins

Plugin Management

Plugins are third-party extensions with restricted permissions.

  • Enable/Disable: node index.js plugins:enable <name>
  • Create new: node index.js plugins:create <name>
  • Validate: node index.js plugins:validate <name>
  • Hot-reload: node index.js plugins:reload <name>

Status Legend

  • ✓ Active - Plugin is enabled and running
  • ○ Loaded - Plugin loaded but not activated
  • ✗ Error - Plugin failed to load or activate
  • - Disabled - Plugin is not enabled
  • ! Changed - Plugin has pending file changes
