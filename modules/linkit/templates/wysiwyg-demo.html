How it works

Inline Autocomplete (Type @):

  • Type @ followed by search text
  • Autocomplete dropdown appears with matching results
  • Use arrow keys (↑/↓) to navigate, Enter to select
  • The link is inserted inline at the cursor position

Link Dialog (Click 🔗 button):

  • Select text and click the Link button
  • Dialog opens with autocomplete field
  • Type to search, select from results
  • Link is applied to selected text

Features:

  • Debounced search (300ms delay)
  • Fuzzy matching across title, body, username
  • Keyboard navigation (↑/↓/Enter/Esc)
  • Works with content, media, and users
