## Managing Content

The content management section is where you create, edit, and organize all content on your site.

### Creating Content

1. Navigate to **Admin > Content**
2. Click **Create** and select a content type
3. Fill in the required fields
4. Choose whether to **Save as Draft** or **Publish**

### Editing Content

To edit existing content:
- Find the item in the content list
- Click the **Edit** icon
- Make your changes
- Save or publish

### Content States

Content can be in one of several states:

- **Draft** - Work in progress, not visible to public
- **Published** - Live on the site
- **Scheduled** - Will publish at a future date
- **Archived** - No longer active but retained

### Bulk Actions

Select multiple items to perform bulk operations:
- Publish/unpublish multiple items
- Delete selected content
- Change content state
- Export to different formats

### Filtering and Search

Use the filter sidebar to:
- Filter by content type
- Filter by state (draft, published, etc.)
- Filter by author
- Search by title or content

### Tips

- Use **Save as Draft** to work on content over multiple sessions
- Set publish dates to schedule content in advance
- Use revisions to track changes and revert if needed
- Add tags and categories to make content easier to find
