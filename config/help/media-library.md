## Media Library

The media library is where you manage all images, documents, and other files used on your site.

### Uploading Files

Upload files to your media library:

1. Go to **Admin > Media**
2. Click **Upload**
3. Drag and drop files or click to browse
4. Files are automatically processed and added

### Supported File Types

- **Images** - JPG, PNG, GIF, SVG, WebP
- **Documents** - PDF, DOC, DOCX, XLS, XLSX
- **Video** - MP4, WebM, OGG
- **Audio** - MP3, WAV, OGG

### Organizing Media

Keep your library organized:

- **Folders** - Create folders to group files
- **Tags** - Add tags to files for easy searching
- **Alt Text** - Add descriptions for accessibility
- **Metadata** - Track file information

### Using Media in Content

Add media to your content:

1. In the content editor, click **Add Media**
2. Select from library or upload new
3. Choose size and alignment
4. Add caption if desired
5. Insert into content

### Image Styles

Automatically generate different image sizes:
- **Thumbnail** - Small preview (150x150)
- **Medium** - Content images (600x400)
- **Large** - Full-width images (1200x800)
- **Custom** - Define your own sizes

### File Management

Manage your media files:
- **Edit** - Update metadata and tags
- **Replace** - Upload new version
- **Delete** - Remove from library
- **Download** - Save to your computer

### Search and Filter

Find files quickly:
- Search by filename
- Filter by file type
- Filter by upload date
- Browse by folder or tag

### Best Practices

- Name files descriptively before uploading
- Add alt text to all images for accessibility
- Use appropriate file formats (WebP for photos, PNG for graphics)
- Organize files into folders
- Delete unused files to save space
- Optimize images before uploading

### Bulk Operations

Work with multiple files:
- Select multiple files
- Add tags to all selected
- Move to different folder
- Delete in bulk

### Tips

- Compress large images before uploading
- Use consistent naming conventions
- Add alt text for SEO and accessibility
- Create folders for different content types
- Clean up unused media regularly
